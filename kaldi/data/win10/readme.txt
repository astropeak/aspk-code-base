All sound files in this directory or sub directory are recored by me on win10 computer.
The sound file are in m4a format, and it can be converted to wav by ffmpeg.
 ffmpeg -i 10101010.m4a -ac 1 -ar 8000 10101010.wav

The train_fmllr dir is used to train fmllr transfromation file. sound files that are in this directory can be used to test it they can be recognized.
